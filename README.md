# react-todo
Learning React: basic todo app in React and Flux

#### Demo page
https://rawgit.com/deepakanand/react-todo/master/public/index.html


#### Run

```
npm install
npm start
```

#### Develop

Webpack auto-watches client assets.

```
npm install
npm run dev
```

#### Build

Bundle the assets for production.

```
npm run build
```
